# Food-Babel-App

# Tech Stack
 
  - Reactjs
  - TailwindCss
  - parcel and babel
  - using Swiggy Public Api

# Installation

- Clone the repository
- Go to the project directory and run : `npm install ` for node module.
- And to run project : `npm start`
- Add `Cors` Extention into your browser & you're good to go.
